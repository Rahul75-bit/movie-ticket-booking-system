# 🎬 MovieZone - Movie Ticket Booking System

## Overview

MovieZone is a full-stack movie ticket booking application built using React.js and Spring Boot.

Users can:

* Browse movies
* Select city and theater
* View shows
* Select seats
* Make bookings
* Cancel bookings
* View booking history

Admins can:

* Manage Movies
* Manage Theaters
* Manage Screens
* Manage Shows
* Manage Seats
